# 光と放射、明るさの知覚

# 1.1 放射
放射とは光を含む電磁波または量子の形でのエネルギーの伝達、あるいは伝達されるエネルギーそのもののこと。
以下が本書で扱う放射のだいたいの範囲である。

* 紫外放射:220nm ~ 400nm
* 赤外放射:760nm ~ 830nm
* 可視放射:360nm ~ 830nm


# 1.2 放射測定、単色放射と分光放射測定
**放射測定**:放射のエネルギー、またはそのエネルギーの時間的割合、空間的割合を測定すること

通常の放射、光は、スペクトル分解してみると多くの波長成分からなる、このスペクトルのうち、ごく狭い部分だけをとりだしたものを単色放射という。
この測定を分光放射測定という。
分光放射測定によって各波長ごとのエネルギーを求めると、分光放射分布が得らえれる。
これはとても重要。

## 分布温度
**白熱光源からの放射の可視域の相対分光分布**は分布温度という量で表せる。

**分布温度**:ある温度の黒体からの可視域の相対分光分布が、その白熱光源の可視域の相対分布光源に等しいかまたは近似的に等しいときのその黒体の温度

# 1.3 光、光刺激と明るさの知覚
明るさの知覚は一人一人の人間に特有のものであるので、その大きさを直接比較できない。したがって、
測定の対象はその知覚を引きおこす原因、光刺激である。
言いかえると、測光の対象は可視放射がもつ、目に明るさの知覚を起こさせる属性の大きさ。

## 標準比視感度
標準比視感度:分光応答度の波長に関する相対値の規約

波長555nm(緑)において感度が最大である、それを1とする。555nmから離れるほど感度が下がっていく。

求め方は、二つの波長の単色放射に対して、その二つが等しい明るさをもつように調整する。
明るさ、というのは目で見た明るさ。このデータを多くの人に対してとって平均を求める。

# 1.4 明所視と暗所視
目が暗所に順応しているときは比視感度の最大値の波長は短かいほうにうつる(青よりになる)
